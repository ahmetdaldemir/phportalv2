🧹 Laravel Cache Temizleme

Bu sayfa sunucudaki cache sorunlarını çözmek için hazırlanmıştır.

🔍 Sorun Analizi

Hata: Call to undefined method ReflectionFunction::isAnonymous()

Sebep: PHP versiyonu uyumsuzluğu (PHP 8.0+ gerekli)

💡 Çözüm Yöntemleri

1. Hosting Panelinden PHP Güncelleme

  • cPanel → PHP Selector → PHP 8.0+ seçin
  • Hosting sağlayıcınızdan PHP güncellemesi isteyin

2. Manuel Cache Temizleme

Aşağıdaki dosyaları silin:

bootstrap/cache/*
storage/framework/cache/*
storage/framework/views/*
storage/framework/sessions/*

3. Terminal Komutları

Sunucuda terminal açıp şu komutları çalıştırın:

# Bash/Linux
chmod +x server-fix.sh
./server-fix.sh

# PowerShell/Windows  
powershell -ExecutionPolicy Bypass -File server-fix.ps1

# PHP Script
php clear-cache-manual.php

4. File Manager ile

cPanel File Manager'dan şu dizinleri temizleyin:

  • /public_html/bootstrap/cache/
  • /public_html/storage/framework/cache/
  • /public_html/storage/framework/views/
  • /public_html/storage/framework/sessions/

🚀 Hızlı Çözüm

En hızlı çözüm için aşağıdaki butona tıklayın:

📞 Destek

Sorun devam ederse:

  • Hosting sağlayıcınızla iletişime geçin
  • PHP versiyonunu 8.0+ yapmalarını isteyin
  • Laravel versiyonunu düşürmeyi düşünün

✅ Başarılı Olursa

Cache temizlendikten sonra:

  1. Sayfayı yenileyin (F5)
  2. Browser cache'ini temizleyin (Ctrl+F5)
  3. Telefon listesi sayfasını kontrol edin
